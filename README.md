# Sales Performance Analysis Dashboard

## Project Overview

This project analyzes sales data using Microsoft Power BI to understand revenue performance, product trends, and regional sales distribution. The interactive dashboard provides insights into sales growth, product performance, and key business metrics to support data-driven decision making.

## Objectives

* Analyze overall sales performance and revenue trends
* Identify top-performing products and categories
* Evaluate regional sales distribution
* Provide interactive insights for business decision-making

## Dashboard Features

* Total Sales, Profit, and Order KPIs
* Region-wise and category-wise sales analysis
* Monthly sales trend visualization
* Product performance insights
* Interactive filters and slicers for data exploration

## Tools Used

* Microsoft Power BI
* Data Visualization Techniques
* Data Modeling
* Business Intelligence Reporting

## Key Insights

* Identification of high-performing products and regions
* Clear visualization of monthly sales trends
* Insights into category-level sales contribution
* Data-driven insights for business strategy
